# VS Code Themes by xRiskLab

A collection of carefully crafted VS Code themes designed for optimal readability and developer experience.

## Available Themes

### [Baseline Theme](./baseline-theme/)

A sleek, minimal VS Code theme based on the [Obsidian Baseline theme](https://github.com/aaaaalexis/obsidian-baseline) by Alexis C.

- **Variants**: Dark and Light
- **Features**: Optimized syntax highlighting, semantic highlighting, clean minimal design
- **Marketplace**: [Install from VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=xRiskLab.baseline-theme)

[View Documentation →](./baseline-theme/README.md)

### [Turbo Theme](./turbo-theme/)

A retro GPT 3.5 inspired theme with elegant teal/cyan tones and salmon/tomato accents. Optimized for JSON, TypeScript, and YAML.

- **Variants**: Dark and Light
- **Features**: GPT 3.5 retro aesthetic, optimized for JSON/TypeScript/YAML, semantic highlighting
- **Marketplace**: Coming soon

[View Documentation →](./turbo-theme/README.md)

## Installation

### From VS Code Marketplace

Each theme is available on the VS Code Marketplace. Search for the theme name in VS Code's Extensions view (Cmd+Shift+X / Ctrl+Shift+X) or visit the theme's individual documentation for direct links.

### From Source

1. Clone this repository:
   ```bash
   git clone https://github.com/xRiskLab/vscode-themes.git
   cd vscode-themes
   ```

2. Navigate to the theme directory you want to install
3. Follow the installation instructions in that theme's README
