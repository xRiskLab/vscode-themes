# Turbo Theme for VS Code

A retro GPT 3.5 inspired theme with elegant teal/cyan tones and salmon/tomato accents. Optimized for JSON, TypeScript, and YAML with excellent readability.

## Features

- **Retro GPT 3.5 aesthetic**: Inspired by the classic GPT 3.5 interface with teal/cyan colors
- **Salmon/tomato accents**: Warm color accents for visual interest and differentiation
- **Two variants**: Dark and Light themes with carefully balanced colors
- **Optimized for JSON, TypeScript, YAML**: Enhanced syntax highlighting for these file types
- **Semantic highlighting**: Intelligent token coloring for better code understanding
- **Elegant design**: Clean, modern, and comfortable for extended coding sessions

## Color Palette

- **Teal/Cyan**: `#10a37f`, `#14b8a6`, `#0d9488` - Primary GPT-inspired colors
- **Salmon/Tomato**: `#ff6b6b`, `#ff8c69`, `#ff7f50` - Warm accent colors
- **Backgrounds**: Dark (`#1a1f2e`) and Light (`#fafbfc`) for optimal contrast

## Installation

1. Open VS Code
2. Go to Extensions (Cmd+Shift+X / Ctrl+Shift+X)
3. Search for "Turbo Theme"
4. Click Install
5. Go to Preferences > Color Theme and select "Turbo Dark" or "Turbo Light"

## Syntax Highlighting

### JSON
- **Keys**: Salmon/tomato (`#ff8c69`)
- **String values**: Teal (`#14b8a6`)
- **Numbers**: Teal (`#10a37f`)

### TypeScript
- **Types/Interfaces**: Teal (`#14b8a6`)
- **Functions/Methods**: Salmon (`#ff6b6b`)
- **Keywords**: Salmon (`#ff6b6b`)
- **Properties**: Tomato (`#ff8c69`)

### YAML
- **Keys**: Tomato (`#ff8c69`)
- **Strings**: Teal (`#14b8a6`)
- **Values**: Teal (`#0d9488`)
