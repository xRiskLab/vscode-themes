# Change Log

All notable changes to the Turbo Theme extension will be documented in this file.

## [1.0.0] - 2025-01-23

### Added
- Initial release of Turbo Theme
- Dark theme variant with GPT 3.5 inspired teal/cyan colors
- Light theme variant with elegant color mapping
- Optimized syntax highlighting for JSON, TypeScript, and YAML
- Semantic highlighting support
- Warm amber accents for visual differentiation
- Enhanced UI with better borders and visual hierarchy (light theme)
- Support for unused parameter highlighting
