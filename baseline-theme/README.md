# Baseline Theme for VS Code

A sleek, minimal VS Code theme based on the [Obsidian Baseline theme](https://github.com/aaaaalexis/obsidian-baseline) by Alexis C.

## Features

- **Two variants**: Dark and Light themes with carefully balanced colors
- **Optimized syntax highlighting**: Enhanced Python support with semantic highlighting
- **Clean and minimal**: Inspired by the popular Obsidian Baseline theme
- **Function definitions in grey**: Easy to distinguish function definitions from calls
- **Semantic highlighting**: Intelligent token coloring for better code understanding

## Installation

1. Open VS Code
2. Go to Extensions (Cmd+Shift+X / Ctrl+Shift+X)
3. Search for "Baseline Theme"
4. Click Install
5. Go to Preferences > Color Theme and select "Baseline Dark" or "Baseline Light"
